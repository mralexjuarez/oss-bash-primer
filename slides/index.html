Bash Foundations

What is bash?

Bash is a shell for commmand interpretation.

  • First released on June 8th 1989
  • Bourne-again Shell, replaced Bourne Shell (sh)
  • Default login shell on a lot of systems.

What happens when you login?

Let's go over what happens when you log on to a system. Much of this is covered under INVOCATION in the bash man page.

INVOCATION

  • /etc/profile
  • ~/.bash_profile
  • ~/.bash_login
  • ~/.profile

Enviornment Variables

The printenv command displays enviornment variables.

Example: printenv

Setting & Exporting Variables

Variables set on the command line are only avaliable in that session.

The export command makes the variable available to scripts run from the session.

Example: export

Bash Built-Ins

Bash has a number of commands built in to it.

The type command can let us know if a command is built in or not.

Example: type

I/O Redirection

A fundamental concepts is that of STDIN(0), STDOUT(1), STDERR(2). These are common ideas to most if not all commands

We can begin to harness the scripting power of the "one-liner" when we understand connecting STDOUT of one command to STDIN of another

Redirecting Output to a File

The output of a command will be displayed to STDOUT (this is usually your terminal).

We can redirect this output using either a single (>) or double(>>) angle brackets

Pipes

So knowing about STDIN and STDOUT Pipes are how we connect the output of one command in to the input of another command.

Example: # last | tac

If Loops

Use an if statement if a decision needs to be made


if [ SOME_TEST ]
then
	COMMANDS
fi

Example If Loop


if [ -z $1 ]
then
 echo "Usage Statement $0 requires argument."
fi

Test Operators

In between the square brackets there are various tests we can perform.

-z STRINGTrue if the length of the STRING is zero
-a STRINGTrue if the length of the STRING is zero
-eq, -ne, -lt, -gtEqual, Not Equal, Less Thant, Greater Than
https://www.tldp.org/LDP/Bash-Beginners-Guide/html/sect_07_01.html

For Loops

A for loop is for executing code repeatedly.


for VAR in SEQUENCE
do
	COMMAND1
	COMMAND2
	COMMAND3
done

While Loops

A while loop allows for code to be executed while a test condition returns true.


i=0
while [ $i -lt 4 ]
do
	COMMAND1
	COMMAND2
	COMMAND3
done
